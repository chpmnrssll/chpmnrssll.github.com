Multi-user content management app.

Client app coded with:
* HTML5
* Javascript
* Require.js
* JQuery
* Underscore.js
* Backbone.js
* Marionette.js
* Bootstrap

Server (REST API) coded in:
* PHP
* Slim
* MongoDB

REST API hosted on OpenShift (cloud).
